# Surgafilm21 - Movie Streaming Platform

![Surgafilm21 Preview](https://i.imgur.com/4C0O2d8.png)

**Surgafilm21** adalah platform streaming film modern yang dirancang dengan fokus pada kecepatan akses, antarmuka pengguna yang bersih, dan optimasi performa tinggi. Proyek ini menggabungkan teknologi web terbaru untuk memberikan pengalaman menonton yang mulus di berbagai perangkat.

## 🚀 Fitur Unggulan
- **PWA Ready:** Dapat diinstal di perangkat Android/iOS seperti aplikasi native.
- **High Performance:** Menggunakan server LiteSpeed dan integrasi Cloudflare untuk pemuatan instan.
- **Modern UI:** Antarmuka responsif yang dibangun dengan Tailwind CSS.
- **Interactive Elements:** Slider konten yang halus menggunakan Swiper.js.
- **SEO & Analytics:** Sudah terintegrasi dengan Google Analytics (GA4) dan Google Tag Manager.

## 🛠️ Tech Stack
Proyek ini dikembangkan menggunakan kombinasi teknologi berikut:

* **UI Framework:** [Tailwind CSS](https://tailwindcss.com/)
* **JavaScript Libraries:** [jQuery](https://jquery.com/), [Swiper.js](https://swiperjs.com/)
* **Web Server:** [LiteSpeed](https://www.litespeedtech.com/)
* **Optimization & Security:** [Cloudflare](https://www.cloudflare.com/), [HTTP/3 Protocol](https://en.wikipedia.org/wiki/HTTP/3)
* **Tracking:** Google Analytics 4 & Google Tag Manager
* **Assets:** Font Awesome 6.4.0, cdnjs

## 📦 Struktur Proyek
- `/assets` - Kumpulan file CSS, JS, dan Gambar.
- `/components` - Bagian-bagian UI yang dapat digunakan kembali.
- `/dist` - File siap produksi yang sudah dioptimalkan.

## 📱 Platform yang Didukung
1.  **Web Browser:** Chrome, Firefox, Safari, Edge (Optimized for HTTP/3).
2.  **Mobile:** Progressive Web App (PWA) support.
3.  **Desktop:** Kompatibel dengan aplikasi berbasis Electron/Tauri (Windows & Linux).
